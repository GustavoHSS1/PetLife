🐾 PetLife

Site institucional para clínica veterinária e petshop, desenvolvido durante a trilha Start na Programação da OneBitCode.

📋 Sobre o projeto

PetLife é uma landing page que apresenta os serviços de uma clínica veterinária e petshop, com foco em oferecer uma experiência acolhedora para os tutores de pets. O site conta com seções de apresentação, informações sobre a clínica, o petshop/farmácia e uma área de perguntas frequentes.

✨ Funcionalidades


Início: apresentação da marca com chamada para agendamento via WhatsApp
Clínica: lista de serviços veterinários oferecidos (consultas, vacinação, cirurgias, odontologia, emergência 24h, entre outros)
Farmácia/Petshop: apresentação dos produtos disponíveis
Dúvidas (FAQ): perguntas frequentes com respostas expansíveis
Contato: página dedicada com informações de contato
Link direto para agendamento via WhatsApp


🛠️ Tecnologias utilizadas


HTML5 — estrutura das páginas
CSS3 — estilização e responsividade
JavaScript — interatividade (ex: abrir/fechar perguntas do FAQ)

🚀 Como executar o projeto


1. Clone este repositório: git clone https://github.com/GustavoHSS1/PetLife.git
2. Acesse a pasta do projeto: cd PetLife
3. Abra o arquivo index.html no seu navegador de preferência.

Não é necessário nenhuma instalação de dependências, pois o projeto utiliza apenas HTML, CSS e JavaScript puros.
